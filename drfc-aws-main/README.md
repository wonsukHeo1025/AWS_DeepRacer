# DeepRacer for Cloud

- 본 저장소는 [**DRfC (Deepracer for Cloud)**](https://aws-deepracer-community.github.io/deepracer-for-cloud/)를 파이썬 및 주피터 노트북에서 운영하기 위해 [(주)에이아이캐슬](https://aicastle.io) 에서 재구성 한 것입니다.

## 설치
`pip install -r requirements.txt`
